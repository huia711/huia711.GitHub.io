Blind Visual Motif Removal from a Single Image

2024-01-31 / 4 min read

Title

Blind Visual Motif Removal from a Single Image

Venue

CVPR 2019

Problem

  • Proposing a deep learning based technique for blind removal of such objects.
  • In the blind setting, the location and exact geometry of the motif are unknown.

Motivation

  1. Most watermark removal methods try to reconstruct a single watermarked image relying on user guidance to identify the location of the watermark.
  2. The performance degrades when the inpainting is performed blindly, without the mask of the missing pixels.

Significance/Contribution

  1. Do not require multiple images with the same object, or the exact location of the motif pixels.
    1. Separating the visual motif from the image, by first estimating which pixels contain the motif.
    2. Subsequently reconstructing the latent image.
    3. Reconstructing the visual motif.

Method

Goal:

Using a convolutional neural network (CNN) trained to remove visual motifs embedded in an image.
